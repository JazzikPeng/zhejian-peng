# Zhejian Peng — Personal Site

Personal portfolio for [Zhejian (Jazzik) Peng](https://zhejian-peng.vercel.app).

Built with **Next.js 14**, **Tailwind CSS**, and deployed on **Vercel**.

## Features

- Bento-grid portfolio layout
- Light / dark mode (persisted + system preference)
- Live weather (Palo Alto) and GitHub stats

## Develop

```bash
npm install
npm run dev
```

## Deploy

Production: https://zhejian-peng.vercel.app

Pushes to `main` deploy automatically via Vercel.

## Repo

Source of truth for the live site. Vercel project: `profile`.
